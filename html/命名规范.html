命名规范

HTML

标签

由于 HTML 不区分大小写，按照书写规范，除了 <!DOCTYPE> 声明 ，所有标签都应该小写。 


<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="utf-8">
	<title>hi</title>
</head>
<body>
	hello world
</body>
</html>

属性顺序

书写 html 标签上的属性应该按照以下顺序：

  1. class id
  2. src href
  3. for type
  4. name value
  5. max min pattern
  6. placeholder title alt

<input class="input" id="user" type="text" name="username" value="123456" placeholder="请输入用户名">

CSS

class

类名使用 kebab-case 命名方法。 


.my-class {
	color: red;
}

id

id 使用驼峰式命名。 


#myId {
	color: green;
}

样式顺序

书写 CSS 样式时，应按照一定顺序：

  1. 元素属性：display float
  2. 元素位置：position top right bottom left
  3. 盒式结构：broder border-radius width height
  4. 文本样式：font line-height text-align
  5. 颜色：color background-color
  6. 其他：opacity

div {
	display: block;
	position: absolute;

	top: 0;
	right: 0;
	bottom: 0;
	left: 0;

	border: solid 1px #000;
	border-radius: 5px;
	width: 100%;
	height: 100%;

	font-size: 16px;
	line-height: 24px;
	text-align: center;

	color: #222;
	background-color: #fff;

	opacity: .9;
}

scss

书写 scss 里的样式属性时按照以下顺序：

  1. @extend
  2. @include
  3. 自身属性
  4. 嵌套属性

#container {
	@include my-mixin();

	float: left;

	> div {
		text-align: center;
	}
}

16 进制颜色

使用 16 进制颜色的时候尽量遵循以下规则：

  • 使用小写字母
  • 尽量简写为 #xxx

div {
	color: #FFF; // ✖
	color: #ffffff; // ✖
	color: #fff; // ✔
}

javascript

变量

普通变量使用驼峰式命名。 


var myArr = [];
let myArr = [];
jquery 对象前需要添加 $。 

$items = $('.item');

常量

常量全部大写。 


const PASS = 60;

Vue

组件

  • kebab-case
  • 多个单词：避免和 HTML 元素冲突

数据属性

    数据属性使用驼峰式命名。 
    
new Vue({
	data: {
		myDate: 'nothing'
	}
})

php

类名

类名使用大驼峰式，即每个单词首字母大写。 


class MyClass {

}

变量 & 函数

变量和函数名采用驼峰式命名。 


$myArr = [];
function myFunction () {
	
}
