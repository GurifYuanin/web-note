批处理命令

定义

DOS 系统批量处理命令，可以在 window 的命令行执行相应指令。

也可以写入 .bat 文件后双击执行。 

title cmd
echo aaa
pause

基本操作

打印

直接输出。

echo 被输出的字符串

暂停执行

使用 pause 暂停执行，按任意键继续执行。

pause

注释

使用 :: 或者 REM 进行单行注释。

:: aaa
REM aaa

样式

设置控制台的样式。

背景色和前景色

使用 color 背景色代号前景色代号 来设置。

代号颜色
0黑色
1蓝色
2绿色
3浅绿色
4红色
5紫色
6黄色
7白色
8灰色
9淡蓝色
A淡绿色
B淡浅绿色
C淡红色
D淡紫色
E淡黄色
F亮白色
默认值为：
color 07

标题

修改控制台标题。

title 控制台标题

文件

切换路径

使用 磁盘号: 来切换磁盘路径。

C:
使用 cd 切换路径。
cd 路径

执行文件

直接执行一个文件。

start 文件路径
使用 A 文件执行 B 文件。
start A B
window 自带的文件有：
  • explorer：资源管理器
  • notepad：记事本
  • calc：计算器
  • cmd：控制台

查找文件的内容

查找文件内指定内容，将查找到的结果打印出来。

find 标识符 "内容" 路径
标识符：
  • /V：反选
  • /C：只打印行数
  • /N：打印行号
  • /I：忽视大小写
  • /OFF[LINE]：不跳过具有脱机属性的文件
