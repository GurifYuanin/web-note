sublime 插件

颜色

ColorHighter：颜色即时显示

ColorHight 为 CSS 颜色即使显示工具，其显示的条件有三种：

  • 默认状态：search_colors_in
  • 悬浮状态：hover
  • 选中状态：selection

ColorPicker：颜色拾取器

安装 ColorPicker 后，默认调出颜色拾取器的默认快捷键为

Ctrl + Shift + C
可以向 Preference >Keys Setting 添加 { "keys": ["ctrl+shift+p"], "command": "color_pick" } 修改快捷键。

空格

空格显示：TrailingSpaces

使用 TrailingSpaces 工具可以让你看见页面上的所有空格，逼死强迫症患者。

空格删除

向 Preference >Keys Setting 添加 { "keys": ["ctrl+shift+t"], "command": "delete_trailing_spaces" } 修改快捷键，此后可以直接使用该快捷键删除所有多余的空格。

提示

边界显示：BracketHighlighter

使用 BracketHighlighter 插件，当光标停留在某个文本上的时候，会自动寻找该文本的前后方框（大括号、小括号、单双引号、标签等）。

文件名提示：AutoFileName

使用 AutoFileName 插件，当输入 url 路径的时候，会自动提示可供选择的文件及目录。

界面

汉化：ChineseLocalizations

使用 ChineseLocalizations 插件可以对工具栏进行汉化。

右键菜单增强：SideBarEnhancements

使用 SideBarEnhancements 为右键菜单添加功能。

主题

在 Package Controler: Install Package 后输入 theme ，自行选择主题下载。

随后在 Preference > Theme 中选择喜欢的主题。

代码

HTML / XML + CSS 代码拾取器：Emmet

Emmet 提供了快捷的代码书写方式，写完一段简短的代码后按 Tab 将自动生成所有代码。 


// 创建 html 模版
!

// 创建 html 标签
标签 #id|.class *数量 关系 标签 {文本}

// 创建 css 样式
css属性首字母: css值首字母

代码校验：SublimeLinter

SublimeLinter 提供了一系列代码校验功能，需要配合 Node 和 Ruby 环境，先下载 SublimeLinter 插件，再安装以下内容：

Sublime Node Ruby 功能 配置文件
SublimeLinter-jsHint npm install -g jshint javascript 代码语法校验 .jshintrc
SublimeLinter-jscs npm install jscs -g javascript 代码书写规范校验 .jscsrc
SublimeLinter-csslint npm install csslint -g css 语法校验 .csslint
SublimeLinter-contrib-scss-lint gem install scss_lint scss 语法校验 .scss-lint.yml
配置方式为，在项目的根目录下创建一个 .xxx 的配置文件（参见上表），写入配置项，该项目的代码将以此文件为配置参数。

javascript 代码格式化：JSFormat

安装 JSFormat 后，再向 Preference > Key Settings 设置快捷键：

{ "keys": ["ctrl+shift+f"], "command": "js_format"}
再使用快捷键格式化所有 javascript 代码。

css 代码格式化：CSScomb

安装 CSScomb 后还需要安装 npm 包 csscomb 


npm install -g csscomb
然后使用快捷键（默认是 Ctrl + Shift + C）使 CSS 属性排序。
Ctrl + Shift + C

代码注释：Docblockr

下载 Docblockr 可以使代码注释更加方便。

Markdown 实时预览：OmniMarkupPreviewer

使用 OmniMarkupPreviewer 通常有两种操作：

  • Ctrl + Shift + M：开启一个小型服务器，用于实时预览 Markdown 文件
  • Ctrl + Shift + E：将 .md 文件渲染 .html 文件
