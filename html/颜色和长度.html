颜色和长度

颜色

CSS 设置颜色值有以下方式：

  • 命名颜色
  • rgb[a]
  • hsl[a]
  • 十六进制

命名颜色

命名颜色，就是使用CSS预先设定好的颜色单词。


<div style="background-color: black">black</div>
CSS2.1 规范定义了17个颜色名：

rgb[a]

rgb 是一种颜色模式，代表将 red(红色) + green(绿色) + blue(蓝色) 三种颜色叠加后得到的颜色。

每种颜色有 8 位深度，也就是 28 = 256 种颜色，其书写方式为： 

color: rgb(-1, -1, -1); /* × */
color: rgb(0, 0, 0); /* √ */
color: rgb(123, 123, 123); /* √ */
color: rgb(255, 255, 255); /* √ */
color: rgb(256, 256, 256); /* × */

rgba 则是在 rgb 的基础上多加一个不透明度通道，即 red(红色) + green(绿色) + blue(蓝色) + alpha 。
alpha 的值为 0~1 , 0 为完全透明， 1 为完全不透明。 

<div style="background-color: rgb(0, 0, 0);">rgb(0, 0, 0)</div>
<div style="background-color: rgba(0, 0, 0, 0.5);">rgba(0, 0, 0, 0.5)</div>
从视觉上看黑色减少了一半，变成了灰色，实际上是将颜色变成半透明，如果这个时候背景颜色是黄色 

body {
	 background-color: yellow;
}
rgba 只能在 IE9+、Firefox 3+、Chrome、Safari 以及 Opera 10+ 中使用，使用的时候需要留意兼容性问题。

hsl[a]

hsl 是另一种颜色模式，代表 Hue(色相) + Saturation(饱和度) + Lightness(明度) 三种方式调控后得到的颜色。

  • Hue 指色相轮的度数，值为 0~360。
  • Saturation 指颜色的鲜艳程度，值为 0~1。
  • Lightness 指颜色的明亮程度，值为0~1。

color: hsl(240, 0.5, 0.5);
hsla 与 rgba 类似，也是添加了一个不透明度通道，只能在 IE9+、Firefox 3+、Chrome、Safari 以及 Opera 10+ 中使用。

十六进制

十六进制与 rgb 实际上是同一种颜色模式，十六进制使用# + 3或6个十六进制数表示。 


color: #123;
color: #123456;
当#后接3个数字时，CSS 会将其进行值复制。
以下两种写法是等价的： 

color: #123;
color: #112233;
十六进制其实也是 rgb 颜色模式。
  • 前两个十六进制数字等价于 rgb 的 r
  • 中间两个十六进制数字等价于 rgb 的 g
  • 后两个十六进制数字等价于 rgb 的 b

    • Web安全色

    理论上讲，rgb 和 十六进制 下通过混合，可以得到 256 * 256 * 256 = 16777216 种颜色，但实际使用中，还需要考虑网页安全色的问题。
    Web安全色：不同的平台（Mac、PC等）有不同的调色板，不同的浏览器也有自己的调色板。这就意味着对于一幅图，显示在Mac上的Web浏览器中的图像，与它在PC上相同浏览器中显示的效果可能差别很大。 选择特定的颜色时，浏览器会尽量使用本身所用的调色板中最接近的颜色。如果浏览器中没有所选的颜色，就会通过抖动或者混合自身的颜色来尝试重新产生该颜色。[2]
    256色里有40种颜色在Macintosh和Windows里显示的效果不一样，所以能安全使用的只有216色。为了尽量让用户看到色彩相同的网页，请尽量使用下面的216色。[2]

    长度单位

    长度单位分为以下两种：

    • 绝对长度单位
    • 相对长度单位

    绝对长度单位

    绝对长度单位有以下：[3]

    • in(inch/英寸)：1英寸 = 2.54厘米
    • cm(厘米)
    • mm(毫米)
    • pt(点)：点是标准印刷单位，1英寸 = 72点
    • pc(派卡)：派卡也是印刷单位，1派卡 = 12点

    相对长度单位

    相对长度单位有以下：

    • px(pixel/像素)：相对屏幕分辨率
    • em：等于父级元素font-size的倍数大小
    • ex：
      • 对于 font-size 属性，等于父元素的x(小写字母)的高度大小
      • 对于非 font-size 属性，如果自身设置了 font-size ，则等于自身元素的x(小写字母)的高度大小；如果没有设置 font-size ，则等于父元素的x的高度大小
    • rem(root em)：等于根元素(html)font-size倍数大小
    • vw / vh：参见 viewport
    所有浏览器的默认大小为 16px ，也就是相当于默认执行了 
    
body {
	font-size: 16px;
}
    rl_demo.html源代码： 
    
<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>rl demo</title>
	<style>
		div {
			margin: 50px;
			padding: 0;
			border: none;
		}
	</style>
</head>
<body>
	<!-- px -->
	<div style="font-size: 50px;">
		<div style="
			height: 50px;
			font-size: 50px;
			background-color: #ff0;
			line-height: 50px;
		">
			哈(px)
		</div>
	</div>
	<!-- em -->
	<div style="font-size: 50px;">
		<div style="
			height: 1em;
			font-size: 1em;
			background-color: #f0f;
			line-height: 1em;"
		>
			哈(em)
		</div>
	</div>
	<!-- ex -->
	<div style="font-size: 50px;">
		<div style="
			height: 1ex;
			font-size: 1ex;
			background-color: #0ff;
			line-height: 1ex;
		">
			哈(ex)
		</div>
	</div>
	<!-- rem -->
	<div style="font-size: 50px">
		<div style="
			height: 1rem;
			font-size: 1rem;
			background-color: #00f;
			line-height: 1rem;
		">
			哈(rem)
		</div>
	</div>
</body>
</html>
    • px 直接使 height 和 font-size 都固定为 50px
    • em 参照父元素的 font-size ，使得 height 和 font-size 都为 50px
    • ex 参照父元素的 x 的高度，使得 height 和 font-size 约等于 25px (不同字体有所差异)
    • rem 参照 body 元素的 font-size ，因此 height 和 font-size 都为 16px
    rem 是CSS3新增的单位，不兼容IE8及以下。
