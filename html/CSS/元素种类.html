元素种类

分类

CSS中，元素可以从两个方面进行分类。

  • 一是从是否可替换区分，分为可替换元素与不可替换元素
  • 二是从显示效果区分，分为block、inline、inline-block、run-in等
我们可以用 display 属性来进行修改，这里只解释第二种分类方式。

block

特点

  1. 每个元素占整一行。
  2. 高度 (height) 、宽度 (width) 、行高 (line-height) 以及内外边距 (padding + margin) 都可设置。

常见元素

  1. div
  2. p
  3. h1~h6
  4. table
  5. label
  6. form
  7. table
  8. ol
  9. ul
  10. li
  11. pre

inline

特点

  1. 只占领元素自身的大小，不同元素可以在同一行上。
  2. 高度 (height) 、宽度 (width) 及顶部和底部外边距 (margin-top + margin-bottom) 不可设置。

常用元素

  • a
  • b / strong
  • em
  • i
  • u
  • q
  • s / strike
  • span
  • img
  • textarea
  • code
  • sup / sub

inline-block

特点

  1. 只占领元素自身的大小，不同元素可以在同一行上。
  2. 高度 (height) 、宽度 (width) 、行高 (line-height) 以及内外边距 (padding + margin) 都可设置。

多余空格

若元素成为了 inline-block 元素，那么如果： 


<div style="display: inline-block; border: solid white 1px;">hello</div>
<div style="display: inline-block; border: solid white 1px;">world</div>
hello
world
hello 和 world 会有一个空格，因为实际上写的代码是这样的： 

<div style="display: inline-block; border: solid white 1px;">hello</div>回车
<div style="display: inline-block; border: solid white 1px;">world</div>
解决方法一：不要这个回车，特别是当多个 inline-block 元素时，可以这样写： 

<ul class="inline-block-ul">
	<li>
		要点一
	</li><li>
		要点二
	</li><li>
		要点三
	</li><li>
		要点四
	</li>
</ul>
<style>
.inline-block-ul > li {
	display: inline-block;
}
</style>
  • 要点一
  • 要点二
  • 要点三
  • 要点四
解决方法二：不要闭合标签。 

<ul class="inline-block-ul">
	<li>要点一
	<li>要点二
	<li>要点三
	<li>要点四
</ul>
  • 要点一
  • 要点二
  • 要点三
  • 要点四
解决方法三：让父元素 letter-spacing 活 word-spacing 为负数，子元素不变。letter-spacing 表示字母间距，word-spacing 表示单词间距。 

<ul class="inline-block-ul letter-spacing">
	<li>要点一</li>
	<li>要点二</li>
	<li>要点三</li>
	<li>要点四</li>
</ul>
<style>
	.letter-spacing {
		letter-spacing: -3px;
	}
	.letter-spacing > li {
		letter-spacing: 0px;
	}
</style>
  • 要点一
  • 要点二
  • 要点三
  • 要点四

设置行列宽高

通过 grid-template-rows 来设置每一行的高度，通过 grid-template-columns 来设置每一列的宽度。 


selector {
	grid-template-rows:  数量 fr|长度单位;
	grid-template-columns: 数量 fr|长度单位;
}
既可以用原有的长度单位，比如 px，也可以用适用于网格布局的新单位 fr ，fr 表示获得剩余空间的等分数，比如： 

<div style="height: 100px; display: grid; grid-template-rows: 1fr 2fr 1fr;">
	<div style="background-color: #0f0f0f;">1fr</div>
	<div style="background-color: #8f8f8f;">2fr</div>
	<div style="background-color: #3f3f3f;">1fr</div>
</div>

2fr

repeat 方法

repeat 方法用于生成多个相同的值。
repeat(重复次数, 值)
比如以下写法是等价的： 

/* 1fr 2fr 的格式连续出现了 3 次 */
selector {
	grid-template-rows: 4fr 1fr 2fr 1fr 2fr 1fr 2fr;
}
/* 使用 repeat 方法 */
selector {
	grid-template-rows: 4fr repeat(3, 1fr 2fr);
}

自动设置行列宽高

当没有显示地指定行列宽高(grid-template-rows 或 grid-template-columns)的时候，这时网格布局就会依赖另一个属性来设置宽高：grid-auto-rows 或 grid-auto-columns。 


<div style="display: grid; grid-template-columns: 1fr 1fr; grid-template-rows: 50px 50px; grid-auto-rows: 100px;">
	<div style="background-color: #888">1</div>
	<div style="background-color: #888">2</div>
	<div style="background-color: #888">3</div>
	<div style="background-color: #888">4</div>
	<div style="background-color: #222">5</div>
</div>

minmax 方法

minmax 方法可以用在 grid-auto-rows 或 grid-auto-columns 属性上，用于定义网格元素最小和最大允许的宽高（由于元素内的东西大小不定，网格元素的宽高总是大小不定）。
minmax(最小值, 最大值) 

<div style="display: grid; grid-auto-rows: minmax(20px, auto);">
	<div style="background-color: #666"></div>
	<div style="background-color: #888"><p>段落1</p><p>段落2</p><p>段落3</p></div>
</div>

段落1

段落2

段落3

第一个元素内没有东西，高度应该为 0 ，但 minmax 规定最小高度为 20px，所以最终高度为 20px 。第二个元素内有 3 个 p 标签，高度大于 20px，此时由于最大高度为 auto ，因此正常显示。

使用 / 来简写

总是要写 grid-row-start, grid-row-end 等过于繁琐，网格布局提供了合并属性 grid-row 或 grid-column，可以直接使用 / 方式来设置开始和结束编号：
grid-row: 开始编号/结束编号
grid-column: 开始编号/结束编号

<div style="display: grid; grid-template-columns: repeat(3, 1fr)">
	<div style="grid-column: 1/4">1/4</div>
	<div>5</div>
	<div>6</div>
	<div>7</div>
</div>

横跨 n 个轨道

除了设置开始编号和结束编号，另一种表示写法是从轨道 k 开始横跨 n 个轨道：
grid-row: 开始编号/span 长度
grid-column: 开始编号/span 长度
效果等同于 grid-row: 开始编号/开始编号+长度。

嵌套

元素间的嵌套遵循以下：

  • 块级元素可以包含内联元素和某些块级元素
  • 内联元素可以包含内联元素，但不能包含块级元素
以上只是笼统的说法，实际情况复杂许多。

p

p 内部不能嵌套块级元素。 


<p>
	<div>hello world</div>
</p>
<!-- 实际上会被渲染为 -->
<p></p>
<div>hello world</div>

a

a 内部可以嵌套块级元素，以下会正常渲染。 


<a href="javascript:void(0)"><div>hello world</div></a>
