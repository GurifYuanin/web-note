CSS最新特性

选择器

伪类

:has()

根据其后代选择一个元素。比如下面的例子中，只有第二个 section 中的 b.jpg 图片样式生效 

<section>
  <img src="a.jpg" alt="">
</section>
<section>
  <h2>Title</h2>
  <img src="b.jpg" alt="">
</section>
<style>
  section:has(h2) img {
    border: solid 1px black;
  }
</style>

:is() / :where()

这两个伪类用来选择多个选择器，比如下面的情况是等价的： 

.class1 div,
.class2 div {}
:is(.class1, .class2) div {}
:where(.class1, class2) div {}
:is() 和 :where() 的不同点在于其选择器权重不同：
  • is：括号中的选择器列表中最高优先级
  • where：总是 0

层叠层

层叠层可以使用 @layer 来对多个选择器进行分组，组与组之间存在优先级关系，后面的组会覆盖前面组的样式。 


@layer reset, base, theme;
@layer reset {
  body {
    padding: 0;
    margin: 0;
  }
}
@layer base {
  h1 {
    font-size: 22px;
    color: #222;
  }
}
@layer theme {
  h1 {
    color: #333;
  }
}

嵌套


.card {
  & h2 {
    color: blue;
  }
  &:hover {
    color: red;
  }
}
等价于预处理器中 

.card {
  h2 {
    color: blue;
  }
  &:hover {
    color: red;
  }
}

表达式

@when/@else

通过 @when/@else 来写类似于 if/else 的条件规则。 


@when media(min-width: 30em) and supports(display: subgrid) {

} @else {

}

颜色

配色方案

color-scheme 属性允许元素指示配色方案，配合选择器在指定范围生效，比如通过伪类 :root 让整个页面都生效。 


:root {
  accent-color: lime;
}

生成颜色

提供了更多定义颜色的方式：

  • hwb()：色相、白度、黑度
  • lab()：亮度、a 值、b 值
  • lch()：亮度、色度、色调
  • color-mix()：将两种颜色混合在一起
  • color-contrast()：从颜色列表中，输出第一个相比对比度最高的颜色
  • color()：指定不同颜色空间中的颜色
