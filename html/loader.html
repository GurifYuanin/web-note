loader

定义

webpack 只认识 javascript 文件，在面对使用 import 导入非 javascript 文件，打包时直接报错。 


					import './style.css';
loader 用于对模块的源代码进行转换。loader 可以使你在 import 或"加载"模块时预处理文件。[1] loader 写在 webpack 配置文件内： 

module.exports = {
	module: {
		rules: [
			{ // 第一条规则
				test: /\.suf$/, // 匹配进行 loader 转换的文件
				use: [ // 匹配到的文件进行数组内所有 loader 转化
					{
						loader: 'loader1', // 第一个 loader 名
						options: { // loader 选项
							option1: true,
							option2: '/'
						}
					},
					'loader2' // 第二个 loader 名
				]
			},
			{
				// 第二条规则
			}
		]
	}
};
多个 loader 时，处理顺序是从下（数组末）往上（数组头）

css

作用：

  • css-loader：将 css 处理为 javascript 认识的文件、处理 css 里的 url、@import 请求
  • style-loader：将 css 装载到 html 中

{
	test: /\.css$/,
	use: ['style-loader', css-loader']
}
