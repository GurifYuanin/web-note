摄像机矫正

作用对象： 摄像机。
作用： 通过参考图像矫正摄像机的方向，以此让模型与图片的背景完美融合。
其基本使用方法为：
  1. 选择“图像”，导入一张参考图像
  2. 选择“矫正”，添加线
  3. 编辑线的位置，使其对准参考图片里的 xyz 轴
  4. 按住shift点击线条，线条依次变色为红、绿、蓝
    • 红色：表示该线条代表 x 轴
    • 绿色：表示该线条代表 y 轴
    • 蓝色：表示该线条代表 z 轴
当矫正状态变成绿色后，表示矫正成功，否则重复添加线和编辑线的工作。

创建背景对象

当不选中摄像机矫正标签的时候，背景会回归 3D 建模的情况，所以可以点击 “创建背景对象” 来创建一个包含图片的背景对象，以此来提供建模的参考。

保护

作用对象： 任意
作用： 锁定物体的位置、缩放、选装属性

合成

作用对象： 任意
作用： 用于控制物体的一些渲染开关，比如投影、折射、反射
  • 摄像机可见：取消勾选后物体本身不会被渲染出来（但不影响其他比如投影的渲染）
  • 投影：
    • 投射投影：取消勾选后自身不会产生投影
    • 接收投影：取消勾选后自身不会接收投影

平滑着色

添加后，物体每三个点之间（三角形）进行平滑渐变着色。如果不添加或者平滑着色角度为 0 ，则每个面着色为纯色。

振动

使物体对象产生位置、旋转、缩放的振动，即 AE 里的 wiggle。

模拟

模拟三维空间的物体，使物体受物理因素（重力、风力）影响。

通用选项

每个模拟标签都有的选项，其含义相同。

力 - 标签

  • 跟随位移：大于等于 0 的值，默认为 0 ，大于 0 时表示物品碰撞时产生吸力而跟随位移
  • 跟随旋转：同上，但产生旋转

缓存

缓存模拟标签的运动，使得下次预览的时候不需要重新（实时）计算。
  • 烘焙全部：即进行缓存
  • 清除缓存

刚体

模拟生活中不会产生变形的坚硬物体。

毛发标签

样条动力学

作用于样条对象，将样条模拟为一根毛发，并产生重力效果。
  • 碰撞：样条是否产生碰撞效果
  • 固定：选择样条上部分点后可以将这些点固定住

毛发碰撞

作用于物体对象，拥有该标签标示会与有“样条动力学”的物体产生碰撞。
