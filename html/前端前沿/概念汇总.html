ReactNative

ReactNative 的架构模型由三部分构成：

  • JS 层：负责构建虚拟 DOM，跨端页面的业务逻辑
  • C++ 桥接层：衔接前端和原生平台（使用 JavaScriptCore 解析执行 JavaScript 代码，通过 MessageQueue 消息队列实现双向通信）
  • Native 层（Java 或 Objective）：负责原生组件的渲染，提供各种原生能力
