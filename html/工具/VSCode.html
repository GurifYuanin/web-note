插件

Code Runner

调用电脑里已经安装好的编译器或解析器（根据 PATH 环境变量）执行代码，快捷键命名为：
  • Run Code：执行代码
  • Stop Code Run：停止代码执行

Remote Development

使用 ssh 连接到远程主机，直接在 vscode 中操作远程主机上的仓库，下载 Remote Development 插件后输入命令： 

> Remote-SSH:
后续操作和使用 shell 连接到远程主机一致，可以输入 user@host ，或直接使用远程主机别名，参见Linux 指令 SSH 连接部分。
Remote Development 目前能在 Visual Studio Code Insiders 上使用。
