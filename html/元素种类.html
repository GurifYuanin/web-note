元素种类

分类

CSS中，元素可以从两个方面进行分类。 一是从是否可替换区分，分为可替换元素与不可替换元素。
二是从显示效果区分，分为block、inline、inline-block、run-in等。我们可以用display属性来进行修改，这里只解释这种分类方式。

block(块级)

特点
1) 每个元素占整一行。
2) 高度(height)、宽度(width)、行高(line-height)以及内外边距(padding + margin)都可设置。
常见元素
1) div
2) p
3) h1 ~ h6
4) hr
5) menu
6) form
7) table
8) ol
9) ul
10) pre

inline(行内)

特点
1) 只占领元素自身的大小，不同元素可以在同一行上。
2) 高度(height)、宽度(width)及顶部和底部外边距(margin-top + margin-bottom)不可设置。
常用元素
1) a
2) b/strong
3) em
4) i
5) u
6) q
7) s/strike
8) span
9) img
10) label
11) textarea
12) select
13) cite
14) code
15) br

inline-block(行内块级)

特点
1) 只占领元素自身的大小，不同元素可以在同一行上。
2) 高度(height)、宽度(width)、行高(line-height)以及内外边距(padding + margin)都可设置。

run-in

直接翻译run-in为跑进，顾名思义，这种显示方式的作用就是当前元素跑进下一个元素内，成为下一个元素的行内部分。
特点
1) 在下一个块级元素(兄弟元素)内部显示，如果下一个元素不是块级元素，则run-in本身成为块级元素。
2) 从原先父元素继承属性，而不从跑进后的父元素继承属性。
