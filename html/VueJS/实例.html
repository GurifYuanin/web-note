实例

定义

每个 Vue 应用都是通过用 Vue 函数创建一个新的 Vue 实例开始的[1]。 


var 实例名 = new Vue(选项对象);
选项对象可以为 vue 实例绑定以下选项属性：
  1. el：元素
  2. data：数据属性
  3. computed：计算属性
  4. methods：方法
  5. filters：过滤器
  6. watch：侦听属性
  7. template：模版
  8. 生命周期钩子
  9. component：组件
  10. mixins：混合
  11. store：将状态（Vuex state）从根组件“注入”到每一个子组件中
  12. router：配置路由

var vm = new Vue({
  el: '选择器',
  data: {
    数据属性名: 数据值
  },
  computed: {
    计算属性名: function() {
      实例名|this.数据属性名...
  }
},
  watch: {
    数据属性名: function(){ // 数据发生改变时触发 }
},
  methods: {
    方法名: function(形参){
      this.$emit('自定义事件名'); // 触发自定义事件
  }
},
  template: '模版内容',
  [before + ]
  created: function(){ },
  mounted: function(){ },
  updated: function() { },
  destroyed: function() { },
  components: { // 局部注册组件
    '组件名': { // 组件选项 }
},
  mixins: [ mixins对象 ]
});
// 可以通过 实例名.$选项名 来引用选项属性
vm.$el === $('选择器')[0]; // => true
其内部工作流程为： 

// 调用 this._init
...
initLifecycle(vm); // 初始化生命周期
initEvents(vm); // 初始化事件
initRender(vm); // 初始化渲染
callHook(vm, 'beforeCreate'); // 调用 beforeCreate 钩子
initInjections(vm); // 初始化 inject
initState(vm); // 初始化状态
initProvide(vm); // 初始化 provide
callHook(vm, 'created'); // 调用 created 钩子
...
if (vm.$options.el) {
  vm.$mount(vm.$options.el); // 挂载到元素上
}

el

el 决定了 vue 实例绑定的 HTML 元素，被绑定的元素可以在 HTML 标签内使用双花括号 {{}} 来对数据、方法进行访问。
元素的选择方式与 jquery 的 $() 方法一样，但只会选出第一个被选中的元素。 


<div id="app">{{message}}</div>
<script>
	new Vue({
		el: '#app',
		data: {
			message: 'hello world'
		}
	});
</script>

data


var 实例名 = new Vue({
	data: {
		数据属性名: 数据值
	}
});

// 组件中，因为组件会被引用多次，所以 data 是一个函数
export default {
	data() {
		return {
			// ...
		};
	}
}

使用数据

在 javascript 中通过 实例名.数据属性名 来引用数据值 

实例名.数据属性名
console.log(vm.message);
在 html 中通过双花括号包裹（“Mustache”语法 ）数据属性名来引用数据值，该过程称为 插值 。 

<标签>{{数据属性名}}</标签>
<div>{{message}}</div>

转义

其插入的值是转义后的，不会被当做 html 标签被浏览器进行渲染。 如果想取消这种转义，则不要在标签内部使用数据，而是使用 v-html 指令 

<div v-html="message"></div>

响应式

绑定的数据是响应式的，修改数据时视图也会进行更新。 

var vm = new Vue({
  el: '#demo',
  data: {
    message: 'hello world'
  }
});
// 设置定时器，2s 后修改 message 的值
setTimeout(function(){
  vm.message = 'hi !!!';
}, 2000);
如果想取消自动更新，可在标签上添加指令 v-once 

<标签 v-once>{{数据属性名}}</标签>
<div v-once>{{message}}</div>

computed

如何使用

computed 可以是一个带有返回值的函数，通过访问其他数据进行运算 ，最终返回一个值。 

new Vue({
	computed: {
		计算属性名: function() {
			return ...
		}
	}
});
计算属性也可以是一个对象，为其设置 getter 和 setter ，访问时获得 getter 的返回值，修改时调用 setter ，将赋予的值作为形参传入。 

new Vue({
	data: {
		name1: 'abc',
		name2: 'def'
	},
	computed: {
		计算属性名: {
			get: function() {
				return this.name1 + ':' + this.name2;
			},
			set: function(val) {
				this.name1 = val.split(':')[0];
				this.name2 = val.split(':')[1];
			}
		}
	}
});

缓存

计算属性同样是响应式的，只有当计算属性内的相关依赖改变时才会重新执行一遍函数： demo 

<div id="demo">{{getMessage}}</div>
<script>
	var vm = new Vue({
	  el: '#demo',
	  data: {
	    message: 'hello world'
	  },
	  computed: {
	    getMessage: function() {
	      return this.message;
	    }
	  }
	});
</script>

不是函数

计算属性的性质类似于数据属性，不是一个函数，如果在 html 中把它当做方法来调用则会报错。

原理

在创建 computed 属性时，会循环所有计算属性，为每一个属性创建一个 Watch 对象，通过 defineProperty 监听
  • 在 get 中，（组件挂载前）进行依赖收集
  • 在 set 中，执行计算方法

methods


<标签>{{方法名(参数)}}</标签>
<script>
	new Vue({
		methods: {
			方法名: function(形参) {
				...
			}
		}
	});
</script>
方法的调用与函数调用无差异：输出当前时间 

<div id="demo">{{getDate()}}</div>
<script>
	var vm = new Vue({
		el: '#demo',
		methods: {
			getDate: function() {
				return new Date();
			}
		}
	});
</script>

filters

filters 与方法类似，不同的是其调用方式。

创建过滤器


// 全局创建过滤器
Vue.filter('过滤器名', function(参数) {});
// 局部创建过滤器
new Vue({
	filters: {
		过滤器名: function(参数) {}
	}
});

使用过滤器

可以在 {{}} 内或 v-bind 内使用过滤器，使用 | 将参数与过滤器隔开，参数将传入过滤器作为第一个参数，并且支持链式写法。 


{{'hello' | reverse | toUpperCase}}

watch

watch 是对数据属性的侦听，侦查属性与数据属性同名，当数据属性改变时触发，传入改变后的值作为形参。 


new Vue({
	data: {
		数据属性名: 值
	},
	watch: {
		数据属性名: function(val) {
			...
		}
	}
});
				

<div id="demo">{{message}}</div>
<script>
	var vm = new Vue({
		el: '#demo',
		data: {
			message: 'hello world'
		},
		watch: {
			message: function(val) {
				console.warn('你把 message 改为' + val);
			}
		}
	});
</script>

template

替换

模版值将替换掉挂载元素内的所有内容。 

new Vue({
	template: '模版值'
});
				

<div id="demo">啊啊啊我不甘心！！！</div>
<script>
	new Vue({
		el: '#demo',
		template: '<div>任你写什么都会被我替换掉娃哈哈</div>'
	});
</script>

根标签

模版值不能是纯文本节点，必须带有一个根标签，否则会报错 更多可参见模版部分。

生命周期钩子

每个 Vue 实例在被创建时都要经过一系列的初始化过程[1]，该阶段将运行一些生命周期的钩子，此后在不同的生命周期也会有不同的钩子被调用。

component

component 用于局部注册组件，详见组件的局部注册部分。

mixins

mixins 是对 vue 实例选项的一种预设，详见 mixins 。

store

使用 store 选项将状态（Vuex state）注入到每一个子组件中，详见 Vuex 。 


Vue.use(Vuex);
new Vue({
	store,
	component: {
		myComponent: {
			// 组件选项
		}
	}
});

router

使用 router 选项配置路由，详见 vue-router 。 


new Vue({
	router: new VueRouter({
		routes: [
			{
				path: '/foo',
				component: {
					// 组件选项
				}
			}
		]
	})
});
