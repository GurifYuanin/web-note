mixins

定义

mixins 可以包含任意组件选项，当被引入作为参数时，将与其他参数进行混合。

使用


// 局部使用 mixins
// 创建一个 mixins 对象
var myMixins = {
	data: {
		message: 'hello world'
	}
};
// 在组件中使用 mixins
Vue.Component('my-component', {
	extends: myMixins,
	mixins: [myMixins]
});
// 在创建 vue 实例时使用 mixins
new Vue({
	extends: myMixins,
	mixins: [myMixins]
});
// 在创建构造器时使用 mixins
Vue.extend({
	extends: myMixins,
	mixins: [myMixins]
});

// 全局注册 mixins ，其后所有创建的 vue 实例都会被影响
Vue.mixin({});

混合方式

当有同名选项时，按以下方式混合：

  • 数据：同名数据 mixins 的数据被覆盖
  • 钩子：同名钩子被合并为数组，触发时都会被调用，mixins 的钩子优先调用
  • 其他：同名键名 mixins 的键值对被覆盖
当有 extends 、 mixins 和其他选项参数时候，其优先级为：
  1. 其他选项参数
  2. mixins
  3. extends

var mix1 = {
	data: function() {
		return {
			name: 'this is extends'
		};
	}
};
var mix2 = {
	data: function() {
		return {
			name: 'this is mixins'
		};
	}
};
var vm = new Vue({
	extends: mix1,
	mixins: [mix2],
	data: {
		name: 'this is other'
	}
});
console.log(vm.name);
