CSS3

定义

CSS(Cascading Style Sheets)：层叠样式表，是一种 样式表 语言，用来描述 HTML 或 XML（包括如 SVG、XHTML 之类的XML 分支语言）文档的呈现。[1]

  • CSS3 是最新的 CSS 标准
  • Safari、Chrome、Firefox 和 Opera几乎支持全部的 CSS3 特性。 IE10+ 也基本上支持 CSS3 特性

圆角

border 用于设置圆角，遵循值复制原则：

  • border: 左上角 右上角 右下角 左下角

border-radius: 20%;

背景图片

  • background-image: url('img1'), url('img2')
  • background-size: width height
  • background-origin: border-box | padding-box | content-box
  • background-clip: border-box | padding-box | content-box

background-position: left top;
background-repeat: no-repeat;
background-image: url(../images/rock.jpg);/* 背景图片(多张图片时重叠) */
background-size: 100px 100px; /* 背景图片宽高 */
background-origin: content-box; /* 背景图片绘制起点 */
background-clip: content-box; /* 背景图片画布 */
需要注意的是，如果是用外联式引入样式表（link），背景图片的 url 是相对于当前 css 文件，而不是 html 文件。

渐变

linear-gradients 用于设置线性渐变，radial-gradients 用于设置径向渐变：

  • background: linear-gradients(direction, color1 position1, color2 position2, ...)
  • background: repeating-linear-gradients(direction, color1, color2, ...)
  • background: radial-gradients(center, shape, size, color1 position1, color2 position2, ...)
  • background: repeating-radial-gradients(center, shape, size, color1 position1, color2 position2, ...)
样式值的意思为：
  • direction：left | top | right | bottom | <number>deg，线性渐变方向
  • color：任意颜色（可以带透明度）
  • position：任意长度，颜色所在位置
  • center：中心点所在位置，参考 background-position
  • shape：circle | ellipse，径向渐变形状
  • size：closest-side | farthest-side | closest-corner | farthest-corner，径向渐变范围大小

background: linear-gradient(90deg, #000, #fff); /* 线性渐变 */
background: radial-gradient(circle, #000, #fff); /* 径向渐变 */

投影

text-shadow 用于设置文字投影，box-shadow 用于设置元素投影。

  • text-shadow：x-offset y-offset blur color
  • box-shadow：x-offset y-offset blur spread color inset
样式值的意思为：
  • x-offset：投影的 x 偏移（向右为正方向）
  • y-offset：投影的 y 偏移（向下为正方向）
  • blur：投影的模糊度
  • spread：投影的尺寸
  • color：投影颜色
  • inset：切换为内投影

text-shadow: 5px 5px 5px gray; /* 文字投影 */
box-shadow: 5px 5px 5px 5px gray; /* 外部投影 */
box-shadow: 5px 5px 5px 5px gray inset; /* 内部投影 */

@keyframes

使用 @keyframes 预设一个动画，再将这个动画添加到 css 属性 animation 上，元素将立即执行这个动画。

点击按钮

/* 设置一个动画 */
@keyframes bigger{
	from {
		width: 100px;
		height: 100px;
	}
	to {
		width: 200px;
		height: 200px;
	}
}
div {
	animation: bigger 1s; /* 添加后立即执行 */
}

flex

flex box 是一种当页面需要适应不同的屏幕大小以及设备类型时确保元素拥有恰当的行为的布局方式。其分为两部分：

  • flex container：容器，即父元素，所有子元素自动成为容器的成员
  • flex item：项目，即容器的成员
设置 flex container 后，flex item 的以下属性将失效：
  • float、clear
  • vertical-align

容器（container）

容器可以设置的属性：

  1. flex-direction：row | row-reverse | column | column-reverse 项目排列方向
  2. flex-wrap：nowrap | wrap | wrap-reverse 项目换行方式
  3. justify-content：flex-start | center | flex-end | space-between | space-around 项目水平对齐方式
  4. align-items：flex-start | center | flex-end | baseline | stretch 单行时，项目垂直对齐方式
  5. align-content：flex-start | center | flex-end | space-between | space-around | stretch 多行时，项目垂直对齐方式

display

通过修改 display 来设置为 flex 容器： 

<div style="display: flex">块级元素</div>
<span style="display: inline-flex">行内元素</span>

flex-direction

通过修改 flex-direction 来设置项目的排列方向：
  • row：水平从左到右，默认
  • row-reverse：水平从右到左
  • column：垂直从上到下
  • column-reverse：垂直从下到上

<div style="display: inline-flex; flex-direction: row;">
	<div style="width: 20%; margin: 2.5%">item1</div>
	<div style="width: 20%; margin: 2.5%">item2</div>
	<div style="width: 20%; margin: 2.5%">item3</div>
	<div style="width: 20%; margin: 2.5%">item4</div>
</div>

项目（items）

项目可以设置的属性：

  1. order: <integer> 项目位置的权重值
  2. flex-grow：<number> 项目缩放的权重值
  3. flex-shrink：<number> 项目缩小的权重值
  4. flex-basis：长度 | auto 项目自定义宽度
  5. align-self：auto | flex-start | flex-end | center | baseline | stretch 项目自定义垂直对齐方式

order

默认为 0 ，权重越小越靠前。 

<div style="display: inline-flex;">
	<div style="order: 1">1</div>
	<div style="order: 0">0</div>
	<div style="order: -1">-1</div>
</div>

flex-grow

默认为 0 ，即不缩放。在有剩余空间的情况下：
  • <= 0 时表示不缩放
  • > 0 时表示缩放，值的大小表示缩放的权重（瓜分剩下的空间）

<div style="display: inline-flex;">
	<div style="flex-grow: 0">0</div>
	<div style="flex-grow: 2">2</div>
	<div style="flex-grow: 1">1</div>
</div>

flex应用

水平垂直居中

通过设置 justify-content 和 align-items 属性来让唯一的子元素在 flex 容器内水平垂直居中。 


<div style="display: flex; justify-content: center; align-items: center; height: 200px; width: 200px; background-color: gray">
	<div style="background-color: black; color: white;">水平垂直居中元素</div>
</div>
水平垂直居中元素

水平方向自适应

有时需要实现一个布局：单列，3 个元素，左右元素是固定宽度（比如 200px），中间元素自适应宽度。 


<div style="display: flex;">
	<div style="width: 200px; background-color: #888; color: white;">200px</div>
	<div style="flex-grow: 1; background-color: #222; color: white;">自适应</div>
	<div style="width: 200px; background-color: #888; color: white;">200px</div>
</div>
