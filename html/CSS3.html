伪类伪元素

定义

CSS(Cascading Style Sheets)：层叠样式表，是一种 样式表 语言，用来描述 HTML 或 XML（包括如 SVG、XHTML 之类的XML 分支语言）文档的呈现。[1]

  • CSS3 是最新的 CSS 标准
  • Safari、Chrome、Firefox 和 Opera几乎支持全部的 CSS3 特性。 IE10+ 也基本上支持 CSS3 特性

新特性

border-radius(圆角)


							border-radius: 20%; /* 圆角 */

background(背景图片)

  • background-image
  • background-size
  • background-origin
  • background-clip

							background-position: left top;
							background-repeat: no-repeat;
							background-image: url(../images/岩石.jpg);/* 背景图片(可同时设置多个) */
							background-size: 100px 100px; /* 背景图片宽高 */
							background-origin: content-box; /* 背景图片绘制起点 */
							background-clip: content-box; /* 背景图片画布 */

Gradients

  • Linear Gradients(线性渐变)
  • Radial Gradients(径向渐变)

							background: linear-gradient(90deg, #000, #fff); /* 线性渐变 */
							background: radial-gradient(circle, #000, #fff); /* 径向渐变 */

shadow

  • text-shadow(文字投影)
  • box-shadow(盒子投影)

文字投影

							text-shadow: 5px 5px 5px #000; /* 文字投影 */
							box-shadow: 5px 5px 5px #000; /* 盒子投影 */

text-overflow

abcdefghijklmnopqrstuvwxyz

							text-overflow: ellipsis; /* 文字溢出部分用"..."代替 */
							overflow: hidden;

transform

旋转90deg
偏移100px
XY缩放2倍
XY倾斜10deg
X轴旋转45deg(3d)
XYZ轴偏移100px

							transform: rotate(90deg);
							transform: translate(100px);
							transform: scale(2, 2);
							transform: skew(10deg, 10deg);
							ransform: rotateX(45deg);

transition

悬浮我

							transition: width 1s, height 1s; /* 打开 width 和 height 动画开关 */
							div:hover {
								width: 200px;
								height: 200px
							}

@keyframes

刷新后变大

							@keyframes bigger{
								from {
									width: 100px;
									height: 100px;
								}
								to {
									width: 200px;
									height: 200px;
								}
							}
							animation: bigger 1s; /* 添加后执行 */
