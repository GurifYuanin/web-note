HOC

定义

在 React 中，HOC（Higher-order function / 高阶组件）指的是接受一个或多个组件作为参数并且返回一个新组件的函数。 


import { Component } from 'react';

// MyComponent 是原组件
class MyComponent extends Component {
  render() {
    return <div>Hello World</div>;
  }
}

// highOrderComponent 是 HOC
function highOrderComponent(Component) {
  return class EnhancedComponent extends Component {
    // 做一些自定义操作
    render() {
      return <Component {...this.props} />
    }
  }
}

// WrappedComponent 是新组件
const WrappedComponent = highOrderComponent(MyComponent);
