renderProps

定义

React 的 renderProps 指的是父组件 通过 props 给子组件传递一个控制渲染的函数，子组件调用该函数来得到渲染结果。 


// 子组件
class ChildComponent {
  render() {
    return (
      <div>
        <h1>Title</h1>
        { this.props.render() }
      </div>
    );
  }
}

// 父组件
class ParentComponent {
  render() {
    return (
      <ChildComponent render={ () => <div>Hello World</div>  } />
    );
  }
}
