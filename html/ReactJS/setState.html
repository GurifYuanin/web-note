setState

不同点

在组件生命周期或合成事件中

  • 异步更新数据
  • 若有多条 setState，进行合并更新

this.setState({ count: this.state.count + 1 });
this.setState({ count: this.state.count + 1 });
this.setState({ count: this.state.count + 1 });
// 等价于
Object.assign(
  previousState,
  { count: previousState.count+ 1 },
  { count: previousState.count+ 1 },
  { count: previousState.count+ 1 },
)

在 setTimeout 或原生事件中

  • 同步更新数据
  • 若有多条 setState，逐条执行
