选择器

定义

CSS 选择器用于对 DOM 树的节点进行选择，即选出特定的标签，以实现后序操作。

  • 通配选择器
  • 元素选择器
  • id选择器
  • 类选择器
  • 属性选择器
  • 包含选择器
  • 兄弟选择器
  • 子选择器

通配选择器

通配选择器选中所有元素。 


/* 所有的元素内外边距设置为 0 */
* {
	margin: 0;
	padding: 0;
}

元素选择器

元素选择器直接使用标签名来获取元素。 


tag-name {
	text-decoration: none;
}

id选择器

id 是赋予元素的唯一标志，一个 id 只能被使用一次，因而使用 id 选择器生成的样式是唯一的。 


#id-name {
	width: 100px;
	height: 100px;
}

类选择器

类(class)可以在多个元素使用，因而可以使用类选择器给一组元素设置同样的样式。 


.class-name {
	width: 100px;
	height: 100px;
}

属性选择器

属性选择器是对已选中元素的二次筛选，选择具有或不具有某个属性或属性值的元素。 


seletcor[attr1][attr2]...[attrn] {} /* 选择同时拥有这些属性的元素 */
selector[attr="value"] {} /* 选择属性值为 value 的元素 */
selector[attr~="value"] {} /* 选择属性值包含 value 的元素 */
selector[attr^="value"] {} /* 选择属性值以 value 开头的元素 */
selector[attr$="value"] {} /* 选择属性值以 value 结尾的元素 */
selector[attr*="value"] {} /* 选择属性值包含 value 的元素 */

包含选择器

包含选择器又称后代选择器，指基于已选择元素，在其子孙节点中选出元素。中间使用 空格 隔开。 


	selector1 selector2 {} /* 在选择器1选出的元素的后代中，使用选择器2进行二次筛选 */

	<!-- HTML -->
	<div>
	<p></p>
		<span>
			<p></p>
		</span>
	</div>
	/* CSS */
	div p {
		width: 100px;
		height: 100px;
	}

兄弟选择器

兄弟选择器指基于已选择元素，在其兄弟节点中选出元素。中间用~隔开。 


selector1 ~ selector2 {}

子代选择器

子代带选择器指基于已选择元素，在其子代节点中选出元素。中间用>隔开。 


selector1 > selector2 {}
