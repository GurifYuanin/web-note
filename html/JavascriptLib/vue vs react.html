vue vs react

在开发习惯上

  • Vue 单文件组件（SFC，即 .vue 文件）使用 template + js + css 的组合模式，对前端开发者来说更易于理解
  • React 偏向于函数式编程，将 html 和 css 融入 js 中，一切皆由 js 控制（all in js）

在响应式的实现上

  • Vue 提供反应式的数据，当数据改动时界面自动更新 => 开发便捷
  • 对于 React 来说，state 是只读的，需要用户给系统一个明确的信号才会重新渲染 => 数据流可见可控、系统稳定

对 TS 的支持上

  • Vue3 之前对 TS 的支持不是很完全，props 和 data 的类型是断层的，比如在 props 类型的声明上： 
    
<script>
  export default {
    props: {
      myData: {
        type: Object,
        required: true
      }
    }
  }
</script>
  • React 通过函数和类来表达组件，所以对 TS 更加契合
