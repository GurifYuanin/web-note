异常

Error

创建一个错误对象，所有错误对象都有三个参数：

  • message：错误消息
  • fileName：报错文件，默认为创建异常所在文件
  • lineNumber：异常所在行数，默认为创建异常时所在行

throw new Error('出错了！');
Uncaught Error: 出错了！

TypeError

类型错误异常。 


undefined.foo();
Uncaught TypeError: Cannot read property 'foo' of undefined

ReferenceError

引用错误异常，当引用了一个不存在的变量时抛出。 


const a = b;
Uncaught ReferenceError: b is not defined

RangeError

范围错误，值不在允许的范围内时抛出。 


const arr = new Array(-1);
Uncaught RangeError: Invalid array length

SyntaxError

语法错误异常。 


#asd
Uncaught SyntaxError: Unexpected identifier

URIError

执行 decodeURI 或 decodeURIComponent 解析出错时抛出。 


decodeURIComponent('%');
Uncaught URIError: URI malformed

EvalError

该异常只能手动抛出，因为执行 eval 出错只会抛出以上的异常。 


throw new EvalError('执行出错');
Uncaught EvalError: 执行出错
