自适应宽度

问题描述

实现这样一个布局：单行三列，左右元素是固定宽度（比如 200px），中间元素自适应宽度。

左右 float 中间 margin

左右两侧的元素 float，中间元素使用 margin 来控制两侧边距，需要注意的是：中间元素要放到后面。 


<div>
  <div style="float: left; width: 200px; background-color: #888; color: white;">200px</div>
  <div style="float: right; width: 200px; background-color: #888; color: white;">200px</div>
  <div style="margin: 0 200px; background-color: #222; color: white;">自适应</div>
</div>
200px
200px
自适应

flex

使用 flex-grow: 1 来让中间元素自动撑开。 


<div style="display: flex;">
  <div style="width: 200px; background-color: #888; color: white;">200px</div>
  <div style="flex-grow: 1; background-color: #222; color: white;">自适应</div>
  <div style="width: 200px; background-color: #888; color: white;">200px</div>
</div>
200px
自适应
200px

calc

通过 css 的 calc 计算出中间元素的宽度。 


<div style="display: flex;">
  <div style="width: 200px; background-color: #888; color: white;">200px</div>
  <div style="width: calc(100% - 400px); background-color: #222; color: white;">自适应</div>
  <div style="width: 200px; background-color: #888; color: white;">200px</div>
</div>
200px
自适应
200px

网格布局

与 flex 实现思路类似，fr 表示剩下空间的等分数。 


<div style="display: grid; grid-template-columns: 200px 1fr 200px">
  <div style="background-color: #888; color: white;">200px</div>
  <div style="background-color: #222; color: white;">自适应</div>
  <div style="background-color: #888; color: white;">200px</div>
</div>
200px
自适应
200px

js 计算宽度


  <div id="js-container" style="display: flex;">
    <div style="width: 200px; background-color: #888; color: white;">200px</div>
    <div id="js-div" style="background-color: #222; color: white;">自适应</div>
    <div style="width: 200px; background-color: #888; color: white;">200px</div>
  </div>
  <script>
    const containerEl = document.getElementById('js-container');
    const divEl = document.getElementById('js-div');
    const { width } = containerEl.getBoundingClientRect();
    divEl.style.width = width - 400 + 'px';
  </script>
200px
自适应

自适应对齐

当只有一行文本时，居中对齐，当大于等于两行时，左对齐。 


.auto-align {
  text-align: center;
  text-align-last: left;
}
