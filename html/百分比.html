百分比

width/height

width 相对于父标签的 width ，height 相对于父标签的 height 。 


<div style="width: 100px; height: 100px; background-color: #ff8;">
	<div style="width: 50%; height: 50%; background-color: #ff0"></div>
</div>

margin/padding

margin 和 padding 相对于父标签的 width。 


<div style="width: 100px; height: 100px; background-color: #ff8;">
	<div style="margin-let: 50%; width: 50px; height: 50px; background-color: #ff0"></div>
</div>

font-size

font-size 相对于父标签的 font-size。 


<div style="font-size: 50px;">父亲
	<div style="font-size: 50%;">孩子
		<div style="font-size: 50%">孙子</div>
	</div>
</div>
父亲
孩子
孙子

line-height

line-height 相对于自身的 font-size。 


<div style="line-height: 100%; font-size: 50px;">父亲
	<div style="line-height: 100%;">孩子
		<div style="line-height: 100%;">孙子</div>
	</div>
</div>
font-size 具有继承性，因此其上相当于 

<div style="line-height: 100%; font-size: 50px;">父亲
	<div style="line-height: 100%; font-size: 50px;">孩子
		<div style="line-height: 100%; font-size: 50px;">孙子</div>
	</div>
</div>
父亲
孩子
孙子

border-width

border-width 无法使用百分比，使用后无效果 


<div style="width: 100px; height: 100px; background-color: #ff8;">
	<div style="width: 50px; height: 50px; background-color: #ff0; border: 100% solid #000"></div>
</div>

top / right / bottom / left

  • top / bottom 相对于非 static 祖先标签的 height
  • left / right 相对于非 static 祖先标签的 width

<div style="position: relative; height: 200px; width: 200px; border: solid 1px #000;">relative
	<div style="position: static; height: 100px; width: 100px; border: solid 1px #000;">static
		<div style="position: absolute; top: 50%; left: 50%; border: solid 1px #000;">absolute</div>
	</div>
</div>
relative
static
absolute

translate / translateX / translateY

xy 轴偏移，百分比相对的都是自身元素的宽高。 


<div style="transform: translate(50%, 50%); width: 100px; height: 100px; background-color: yellow"></div>
