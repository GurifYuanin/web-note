虚拟键盘

收起虚拟键盘后大片空白

问题描述：在高版本的 IOS 中收起虚拟键盘页面会留下大量空白
解决方法：页面滚动一下

document.addEventListener('focusout', () => {
  setTimeout(() => {
    const height = document.documentElement.scrollTop || document.body.scrollTop;
    window.scrollTo(0, height + 1);
    window.scrollTo(0, height - 1);
  }, 20)
});

打开虚拟键盘后覆盖输入框

问题描述：在部分安卓机上，虚拟键盘弹出后会覆盖输入框
解决方法：scrollIntoView() 让输入框出现在浏览器的可见窗口内
