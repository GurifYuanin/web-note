目录

Linux目录结构

/home

用户家目录，比如用户名为 user，则该用户的家为 /home/user。 


# 以 user 用户登录后回家，以下两种方式等价
$ cd ~
$ cd

/etc & /usr/local/etc

/etc 是系统的主要配置文件存放的目录。

  • passwd, group：文件，用户/组的账号和加密后的口令等信息
  • init.d：目录，Linux 初始化
  • ...
/usr/local/etc 是第三方软件配置文件存放的目录。

/usr & /usr/local

Unix Software Resource，Unix 软件资源。 


$ ls /usr
bin etc games include lib lib64 libexec local sbin share src tmp
$ ls /usr/local
aegis bin etc games include lib lib64 libexec sbin share src vim
/usr 和 /usr/local 目录结构类似，/usr 是系统自带的软件，/usr/local 是用户自己下载的第三方软件。

/bin & /sbin/ & /usr/bin

/bin 和 /usr/bin 都是存放指令的目录，/bin 存放一般用户就可以使用的系统指令（cp, ls, ln 等），/sbin 存放 root 用户才能使用的指令（reboot 等），/usr/bin 是第三方软件指令（比如 git）。

shell

profile 用于初始化环境变量，bashrc, shrc, zshrc 等用于配置 shell 的参数，如主题颜色。

  • /etc/profile：任何用户登录后（如 su user）执行的文件
  • /home/`$USER`/.profile：以 $USER 身份登录后执行的文件
  • /etc/bashrc：任何用于登录后启动 bash 后执行的文件
  • /home/`$USER`/.bashrc：以 $USER 身份启动 bash 后执行的文件（每个新窗口都执行一次）
  • /home/`$USER`/.bash_profile：以 $USER 身份启动 bash 后执行的文件（每次会话只执行一次）
  • /home/`$USER`/.zshrc：以 $USER 身份启动 zsh 后执行的文件
bash, sh, zsh 等属于 shell，bash 是 linux 默认 shell，所以最为常用。

/tmp

Temporary files，存放程序所需的临时文件。
