变量

创建变量

直接赋值即可，注意等号左右两边不能有空格。 

$ a=1

使用变量

有四种引用方式：
  • $变量名
  • ${变量名}
  • "$变量名"
  • "${变量名}"
也就是说，使用双引号和不使用引号是等价的，都会解析变量，但如果使用单引号，就不会解析变量。 

$ echo '$a'
$a

内置变量

  • $HOME：指向自己的家，即 ~
  • $PWD：当前路径
  • $OLDPWD：上一次路径，即 -
  • $0：脚本名字
  • $1 $2 ...：执行脚本或函数时的参数
  • $#：参数个数

引用指令

使用 `` 来引用指令。 

$ cd `pwd`/src
# 等价于
$ cd ./src

函数

声明函数

完整的函数格式为： 

function function_name() {
  ...
}
其中可以将可选部分省略： 

function_name {
  ...
}
调用函数的入参使用 $n 的方式，参见上面一节。

使用函数


$ function_name 参数1 参数2 ...

流程控制

条件判断

为了方便起见，只使用双中括号来包裹判断条件，判断符也只使用 == 或 != ： 

$ [[ $a == '1' ]] && echo '$a is 1'
完整的条件判断语法是： 

if [[ $a == '1' ]]; then
  echo '$a is 1'
elif [[ $a == '2' ]]; then
  echo '$a is 2'
else
  echo '$a is not 1 andn 2'
fi
其他常用判断有：
  • [[ -z $xxx ]]：判断 $xxx 是否为空字符串
  • [[ -e $xxx ]]：判断 $xxx 路径是否存在
  • [[ -f $xxx ]]：判断 $xxx 路径的文件是否存在
  • [[ -d $xxx ]]：判断 $xxx 路径的文件夹是否存在

循环


for 变量名 in $数组变量名; do
  echo $变量名
done
