算法规范

通用

使用不同语言，在不同平台上做算法题，可能有不同的规范，但基本都有以下特点：

  • 从标准设备（键盘）输入，最终输出到控制台上
  • 通常入口为 main 函数或 Main 类

Java

使用 Java 做算法题一般都会使用 Scanner 作为读取输入流的类，通常写法为： 


import java.util.Scanner;
class Main {
	public static void main (String[] args) {
		Scanner sca = new Scanner(System.in);
		while (sca.hasNext()) {
			// 读取操作
		}
		sca.close();
		// ...
		System.out.print("your result");
	}
}
Scanner 常用的方法有：
  • hasNext()：判断是否数据
  • nextInt()：读取下一个整型
  • nextFloat()：读取下一个单浮点类型
  • nextDouble()：读取下一个双浮点类型
  • nextLine()：读取一行
然而有时候由于 Scanner 的龟速操作，一些题目无论怎么写都无法 AC，此时可以使用 BufferedReader 类来代替。 

import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
class Main {
	public static void main(String[] args) {
		BufferedReader buf = new BufferedReader(new InputStreamReader(System.in));
		// 进行读取操作
		buf.close();
		// ...
		System.out.print("your result");
	}
}
BufferedReader 常用的方法有：
  • read()：读取一个字符
  • readLine()：读取一行

C++

使用 scanf 进行读取，使用 printf 或 cout 进行输出，最后返回 0。 


#include<iostream>
using namespace std;
int main () {
	int k;
	scanf("%d", &k);
	// ...
	printf("%d", k);
	return 0;
}
