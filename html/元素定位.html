元素定位

定义

定位，即确定元素的位置。
CSS 中使用 position 属性来进行定位方式的设置。

  • static(默认)
  • relative
  • absoulte
  • fixed
  • inherit
除 static 外， relative, absoulte, fixed 都可以使用 top, right, bottom, left 属性进行位置偏移。

static

static 是元素 position 属性的默认值，元素位于文档流(normal flow)中，按照自上到下、从左到右的顺序排放。

relative

relative 从视觉上看，元素相对原先位置进行偏移，但元素依旧占据着原来的位置，元素位于文档流(normal flow)中。


<div style="
	height: 100px;
	width: 100px;
	background-color: #ddd;
">
	<div style="
		height: 50px;
		width: 50px;
		background-color: #000;
		opacity: .5;
		position: relative;
		top: 50px;
	"></div>
	123456789
</div>
其运行结果为：
123456789

absolute

absolute 使元素相对非 static 定位的最近的祖先元素偏移。

  • 元素脱离文档流和文本流，即其后的文本和 static 元素会穿到该元素前面
  • 覆盖 absolute、fixed 和浮动元素，但不覆盖 static 元素

<div style="
	height: 100px;
	width: 100px;
	background-color: #ddd;
	position: relative;
">
	<div style="
		height: 50px;
		width: 50px;
		background-color: #000;
		opacity: .5;
		position: absolute;
		top: 50px;
	"></div>
	<div style="position: static">abc</div>
	123456789
</div>
其运行结果为：
abc
123456789

fixed

fixed 使元素相对视窗偏移，元素脱离文档流和文本流。

上下滚动 

<div id="fixed-block" style="
	display: none;
	background-color: #ddd;
	height: 100px;
	width: 100px;
	text-align: center;
	font-size: 20px;
	line-height: 100px;
	position: fixed;
	top: 0;
	right: 0;
">
	fixed
</div>

位置

如何确定一个元素的位置？关于 Y 轴的有：

  • style.height：元素高度
  • style.top：元素上方偏移量
  • clientTop：元素样式 border-top 的 border-width
  • scrollTop：元素内滚动条的偏移量 / 被隐藏的部分内容的高度
  • offsetTop：和父标签的距离，top + margin-top
  • clientHeight：可视区的高度，height + padding-top + padding-bottom - 滚动条宽度
  • scrollHeight：内容高度 + padding-top + padding-bottom
  • offsetHeight：内容高度（IE、Opera、FF） / height（chrome） + padding-top + padding-bottom + border-top + border-bottom
其中 height 指的是样式上的高度，而"内容高度"指的是元素内的内容的实际高度，比如有 100 行文字，每行文字的 line-height 为 16px ，那内容高度就有 160px，而设置了 height: 100px，那 height 就为 100px。 其他的 left、right、bottom、width 同理。

整体宽高

在这之前需要知道以下元素代表什么：

  • window.document：当前页面文档
  • window.document.documentElement：即 html 元素
    • 
window.document.documentElement === document.getElementsByTagName('html')[0] // => true
  • window.document.body：即 body 元素
    • 
window.document.body === document.getElementsByTagName('body')[0] // => true
名 值 兼容性
window.innerWidth / window.innerHeight window(窗口)可视区宽高 IE9 及以上
window.document.documentElement.clientWidth / window.document.documentElement.clientHeight viewport(视窗) / html 元素宽高 IE7 及以上
window.document.body.clientWidth / window.document.body.clientHeight body 元素宽高 无
getComputedStyle(元素).width / getComputedStyle(元素).height 获得元素的最终（嵌入 + 内联 + 外联）样式宽高 IE9 及以上
